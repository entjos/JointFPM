# Summarises a `JointFPM` objects

This is a summary function for `JointFPM` objects, created with
[`JointFPM()`](https://entjos.github.io/JointFPM/reference/JointFPM.md).
The function improves the readability of the output.

## Usage

``` r
# S3 method for class 'JointFPM'
summary(object, ...)
```

## Arguments

- object:

  An `JointFPM` object.

- ...:

  Other arguments that should be passed to the function.

## Value

No return value, called for side effects.
